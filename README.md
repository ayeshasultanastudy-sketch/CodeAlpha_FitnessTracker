# 🏋️ Fitness Tracker App (CodeAlpha Internship Project)

## 📌 Overview
This is a simple Fitness Tracker web application built using HTML, CSS, and JavaScript. It allows users to track daily fitness activities such as running, walking, workouts, calories burned, and time spent.

---

## ✨ Features
- Add fitness activities with duration and calories
- Track total calories burned
- Track total workout time
- View activity history log
- Visual progress bars for fitness tracking
- Clean and responsive dashboard UI

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

---

## 🚀 How to Run
1. Clone or download the repository
2. Open `index.html` in any browser
3. Start adding your fitness activities
